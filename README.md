# careai

Personal portfolio / profile demo built with Vite + React + TypeScript.

Quick start

- Install dependencies:

```bash
npm install
```

- Run development server:

```bash
npm run dev
```

- Build for production:

```bash
npm run build
```

Project layout

- `src/` — application source (components, pages, assets)
- `public/` — static assets
